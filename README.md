> Uma Landing Page moderna, agressiva e totalmente responsiva desenvolvida para academias que buscam alta conversão e uma identidade visual única.

## 📸 Sobre o Projeto

Este projeto consiste em uma *Single Page Application* (SPA) institucional para academias. O objetivo foi criar uma interface que fugisse do padrão "amarelo e preto", apostando em uma estética **Cyber/High-Tech** (Slate Dark + Neon Lime).

O site foi projetado com foco total na experiência do usuário (UX) e conversão (CRO), direcionando todas as ações (CTAs) diretamente para o WhatsApp do estabelecimento.

## 🚀 Funcionalidades

*   **Design High Voltage:** Identidade visual escura com neons vibrantes e cortes diagonais (Clip-path).
*   **Responsividade Total:** Menu hambúrguer animado e adaptação fluida para mobile, tablets e desktop.
*   **Animações ao Rolar (AOS):** Elementos entram na tela suavemente conforme o usuário navega.
*   **Galeria Bento Grid:** Layout de imagens moderno e assimétrico com efeitos de hover.
*   **Timer de Oferta:** Contador regressivo programável para gerar senso de urgência.
*   **Automação para WhatsApp:**
    *   Formulário de captura que envia os dados (Nome + Objetivo) prontos para o WhatsApp.
    *   Botões de planos com mensagens personalizadas.
    *   Botão flutuante fixo.
*   **FAQ Interativo:** Seção de perguntas frequentes com efeito *accordion*.

## 🛠 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas e leves:

*   **[HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML):** Estrutura semântica e acessível.
*   **[Tailwind CSS](https://tailwindcss.com/):** Framework CSS utilitário para estilização rápida, responsividade e efeitos visuais (via CDN).
*   **[JavaScript (ES6+)](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript):** Lógica para menu mobile, timer, accordion e manipulação de DOM.
*   **[AOS (Animate On Scroll)](https://michalsnik.github.io/aos/):** Biblioteca para animações de scroll.
*   **[FontAwesome](https://fontawesome.com/):** Ícones vetoriais.
*   **Google Fonts:** Tipografias *Exo 2* (Heading) e *Inter* (Body).
